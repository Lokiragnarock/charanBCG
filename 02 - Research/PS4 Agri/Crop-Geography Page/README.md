# Crop & Geography Page — a self-contained R1 artifact (docs only)

> **Filed 2026-07-17.** [[PRD]], [[Spec]], [[Wireframe]] describe a single-file `crop-geography.html` interactive page — status "v1 shipped" per the PRD — but **the built HTML file and its content source (`handoff-summary.md`) are NOT present in this repo**. Either it was built and shipped from a different session/tool and never checked in here, or it predates this vault. Treat as reference/history until the artifact itself is located or rebuilt.

## What it is
A judge-facing single-page interactive: two tiles (Banana crop case / Theni–Tuticorin–Madurai geography case), each opening a themed subsection (banana-yellow / temple-gold) with numbered points, a MOFPI quote, and a 4-fact geography grid. Vanilla HTML/CSS/JS, no build step, Google Fonts CDN only.

## Content overlap with the current site
Its factual content (why banana, why Theni, GT/TNAU/MOFPI citations) overlaps heavily with [[../e-Choupal Banana OS]] and [[../Banana Cluster Spectrum Data]] — those are the current source of truth for the live site. If this artifact gets rebuilt, its numbers should be re-derived from the Ledger, not from its own PRD (the PRD predates the 07-17 pivot and flags 2 facts as UNVERIFIED that the Ledger has since sourced: ripening-chamber count and transport ₹/kg for Theni cluster are now in [[../../PS4 - Link Ledger]] §B).
