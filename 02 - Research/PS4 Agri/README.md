## The audit protocol (run per working session)
1. **Number audit** — any figure used today: is it a Ledger row with 🟢🟡🔴 tag and source? If not, it doesn't exist.
2. **Decision audit** — every choice/kill/pivot made today → dated row in `04 - Decision Log` *the same day*. Memory is not a system.
3. **Spine audit** — did today's evidence change any claim? Re-version the claim (never edit-in-place silently); one line in its history table: what changed, why, triggered-by-what.
4. **Adversary gate** — no claim graduates to `settled` without a logged attack it survived.
5. **Tangent triage** — new tangents get a status (🌱 explore / 🧊 freeze / ⚰️ kill) before session end; untriaged tangents rot into noise.
6. **AI-moments capture** — any "first answer → we pushed → it changed" moment → Working-the-AI table (element 6). Capture at the moment; reconstruction later reads fake.
7. **Citation audit (pre-ship only)** — walk the deliverable; every figure clickable to a source the judge can open.

## Precedence rule (when notes disagree)
**Ledger numbers > Adversary verdicts > Judge-persona scoring > taste.** A claim contradicted by a 🟢 number dies regardless of how much we like it; if numbers can't settle it, the surviving-attack record settles it; if still tied, whichever version scores better against [[../../03 - Personas/The Judge|The Judge]] wins.
